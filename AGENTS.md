# AGENTS.md

## 프로젝트 미션
AI Launch Studio는 **이미 제품을 보유한 사용자를 위한 프로모션 코파일럿**입니다.
MVP 목표는 다음과 같습니다.
- 챗봇/음성 대화로 브리프를 수집한다.
- 시장/경쟁/채널 신호를 반영한 실행 전략을 제시한다.
- 카피/포스터/영상/음성 에셋을 생성한다.
- 결과를 실행 가능한 런칭 패키지로 저장/재조회한다.

## 제품 범위 (현재 고정)
- 핵심 사용자: 이미 제품을 판매/운영 중인 사업자, 브랜드 담당자, 마케터
- 핵심 문제: "이번 주에 어떤 메시지로 어떤 채널에서 무엇을 집행할지"를 빠르게 결정하기 어려움
- MVP 초점: 홍보 전략 + 소재 제작 + 실행 체크리스트
- MVP 제외: 광고 플랫폼 자동 집행, 재고/매출 자동 연동

## 최종 산출물 정의
이 제품의 1차 산출물은 "초기 아이디어 기획서"가 아니다. 아래를 묶은 **실행형 런칭 패키지**다.
1. 시장 스냅샷: 수요 신호/경쟁사 포지셔닝/주의 리스크
2. 포지셔닝/메시지 옵션: 2~3안
3. 주간 실행 플랜: 채널별 액션 체크리스트
4. 크리에이티브 묶음: 카피/포스터/영상 프롬프트/보이스 스크립트
5. 측정 계획: KPI, A/B 테스트 항목, 다음 입력 가이드

## 에이전트 구성 (MVP)
1. 오케스트레이터 에이전트
- 세션 상태 머신 관리
- 게이트 판정과 단계 전이
- 스트리밍 이벤트 송출
- 결과 저장

2. 플래너 에이전트
- 대화형 브리프 슬롯 수집
- 누락 정보 조건부 질문
- 게이트 통과 여부 계산

3. 리서치 에이전트
- 웹 검색/시장 신호 수집
- 경쟁사/가격대/채널 벤치마크 요약
- 근거 출처/신뢰도 메타데이터 반환

4. 전략 에이전트
- 리서치 결과 + 브리프를 결합
- 포지셔닝/메시지/운영 플랜 생성
- 생성할 소재 목록 정의

5. 크리에이티브 에이전트
- 광고 카피/포스터 스펙/영상 프롬프트 생성
- 채널별 변형안 생성

6. 보이스 에이전트
- 내레이션 스크립트
- 톤/속도/강조어/자막 타이밍
- TTS 설정

## 작업 규칙
- 코드 변경 시 관련 문서를 같은 PR에서 함께 수정한다.
- API 변경 시 `docs/api.md`를 즉시 갱신한다.
- DB 변경 시 `docs/db_schema.md`를 즉시 갱신한다.
- 에이전트 책임 변경 시 `docs/orchestrator_design.md`, `docs/prompt_contracts.md`, `docs/voice_agent.md`를 갱신한다.
- 프롬프트/출력 스키마는 버전을 남겨 재현성을 확보한다.

## MVP 완료 기준
- 텍스트/음성 입력 모두에서 동일한 게이트/상태 전이를 보장한다.
- `RUN_RESEARCH -> GEN_STRATEGY -> GEN_CREATIVES` 흐름이 동작한다.
- 전략/카피/포스터/영상/보이스 결과가 구조화 저장된다.
- 실행 이력 조회와 결과 복사/다운로드가 가능하다.

## OpenAI 사용 원칙
- 공식 OpenAI API만 사용한다.
- 에이전트 간 계약은 구조화된 출력으로 고정한다.
- 리서치 단계는 출처 메타데이터를 포함한다.
- 플래너/리서치/전략/크리에이티브/보이스 단계는 스트리밍 이벤트를 제공한다.
- 광고 카피는 안전 필터(과장/금지표현)를 거친 후 제공한다.
- 보이스 톤은 보이스 에이전트 설정으로 일관되게 유지한다.
