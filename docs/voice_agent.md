# 보이스 에이전트 명세

## 1. 목적
영상 결과물의 완성도를 높이기 위해, 일관된 내레이션/보이스 설정/자막 타이밍을 생성한다.

## 2. 입력
- 브리프 슬롯(제품/타겟/채널/목표)
- 리서치 요약(시장 신호/경쟁 포지셔닝)
- 전략 결과(메시지, 포지셔닝)
- 크리에이티브 결과(장면 계획, CTA)

## 3. 출력
- `voiceover_script_short` (10~15초)
- `voiceover_script_long` (20~30초, 선택)
- `tone_profile` (친근/프리미엄/전문가/직설)
- `delivery_profile` (속도/강조어)
- `timing_map` (구간별 시작/종료)
- `subtitle_lines`
- `tts_config` (모델/보이스/포맷)

## 4. 품질 규칙
- 첫 2.5초 훅
- 중간 구간 핵심 베네핏
- 마지막 CTA
- 과장/위험 문구 차단

## 5. 연동
- 출력은 `marketing_assets` 하위에 저장
- 미디어 파이프라인에서 영상과 TTS를 합성
- 필요 시 배경음 ducking 적용
