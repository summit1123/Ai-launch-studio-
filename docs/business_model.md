# 비즈니스 모델 (BM)

## 1. 포지셔닝
"이미 제품을 가진 팀의 주간 홍보 실행을 돕는 AI 프로모션 코파일럿"

## 2. 플랜 구조 (2026-02 기준)
- 현재 출시 상태 (MVP)
  - 스타터(무료)만 즉시 사용 가능
  - 프로/오토파일럿/엔터프라이즈는 오픈 준비중
  - 오픈 준비중 플랜 가격은 정식 출시 시점에 확정
- 스타터 (무료)
  - 계정당 1회 런칭 패키지 생성
  - 브리프 수집 + 시장 스냅샷 + 전략 옵션 + 기본 크리에이티브 제공
  - 결과 다운로드 1회
- 프로 (출시 예정)
  - 런칭 패키지 생성 무제한
  - 영상 생성 크레딧 확대
  - 포스터/영상 프롬프트 커스터마이징
  - 채널별 콘텐츠 변형 자동 생성
  - 결과 저장/재조회/다운로드
- 오토파일럿 (출시 예정, 로드맵)
  - 에이전트 반복 검토 루프 (초안→피드백→개선)
  - 재고/매출 신호 기반 캠페인 제안
  - 온라인 스토어 상품 등록 자동화
  - 주간 성과 리포트 자동 생성
- 엔터프라이즈 (문의)
  - 워크플로우/권한/보안 맞춤 구성
  - 조직 단위 운영 정책
  - SLA 기반 기술 지원

## 3. 수익원
- 구독료(주 수익)
- 미디어 생성 크레딧(부 수익)
- 온보딩/운영 지원 패키지(확장)

## 4. 비용 구조
- 모델 추론 비용(텍스트/이미지/영상/음성)
- 저장소/CDN 비용
- 비동기 작업 인프라 비용

## 5. 단위경제 방향
- 브리프 게이트로 불필요한 생성 비용 절감
- 영상 생성은 옵션화/쿼터화
- 프리셋 재사용으로 재생성률 감소

## 6. 초기 GTM
- 솔로 셀러/소규모 이커머스 운영자부터 공략
- "3분 안에 주간 런칭 패키지" 메시지로 진입
- 저장/재생성/결과물 export로 리텐션 확보

## 7. 범위 원칙 (MVP vs 로드맵)
- MVP 제공 범위
  - 전략 제안 + 소재 제작 + 실행 체크리스트 + 결과 저장/재조회
- MVP 제외 범위
  - 광고 플랫폼 자동 집행
  - 재고/매출 자동 연동
- 로드맵에서 확장할 범위
  - 재고/매출 신호 기반 자동 의사결정
  - 온라인 스토어 등록 자동화
  - 운영 자동화 에이전트 루프
