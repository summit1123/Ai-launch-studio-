# 비즈니스 모델 (BM)

## 1. 포지셔닝
"이미 제품을 가진 팀의 주간 홍보 실행을 돕는 AI 프로모션 코파일럿"

## 2. 요금 가설
- Free
  - 월 실행 횟수 제한
  - 기본 카피/전략 출력
  - 미디어 워터마크
- Pro
  - 실행량 확대
  - 포스터/영상 번들 export
  - 보이스 프리셋/톤 제어
- Team
  - 협업 워크스페이스
  - 승인 플로우
  - 실행 이력/감사 로그

## 3. 수익원
- 구독료(주 수익)
- 미디어 생성 크레딧(부 수익)
- 온보딩/운영 지원 패키지(확장)

## 4. 비용 구조
- 모델 추론 비용(텍스트/이미지/영상/음성)
- 저장소/CDN 비용
- 비동기 작업 인프라 비용

## 5. 단위경제 방향
- 브리프 게이트로 불필요한 생성 비용 절감
- 영상 생성은 옵션화/쿼터화
- 프리셋 재사용으로 재생성률 감소

## 6. 초기 GTM
- 솔로 셀러/소규모 이커머스 운영자부터 공략
- "3분 안에 주간 홍보 플랜" 메시지로 진입
- 저장/재생성/결과물 export로 리텐션 확보
