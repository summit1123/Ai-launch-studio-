# Codex 스킬 운영 가이드

## 1. 현재 세션에서 사용 가능한 스킬
- `openai-docs`: OpenAI 공식 문서 기반 최신 확인
- `skill-creator`: 커스텀 스킬 설계/작성
- `skill-installer`: 스킬 설치/동기화

## 2. 언제 어떤 스킬을 쓰는가
- OpenAI API/모델/정책/스펙이 바뀔 가능성이 있는 질문:
  - `openai-docs` 우선 사용
- 팀 전용 워크플로를 스킬로 굳히고 싶을 때:
  - `skill-creator` 사용
- 외부/공용 스킬을 로컬에 설치할 때:
  - `skill-installer` 사용

## 3. 이 프로젝트에서 추천 커스텀 스킬
1. `launch-brief-planner`
- 슬롯 추출 규칙, 조건부 질문 규칙, 게이트 판정 로직 내장

2. `promo-creative-pack`
- 카피/포스터/영상 프롬프트 출력 스키마 고정

3. `voice-production`
- 보이스 스크립트, TTS 설정, 자막 타이밍 생성 규칙 고정

4. `sora-ops-check`
- Sora 요청 파라미터 검증, 실패 재시도 정책, 결과 품질 점검

## 4. 커스텀 스킬 최소 템플릿
- 목적(언제 쓰는지)
- 입력 스키마
- 출력 스키마
- 실패/폴백 규칙
- 예시 입출력

## 5. 운영 팁
- 스킬 설명은 짧고 명확하게 작성
- 스킬은 하나의 책임만 갖게 유지
- 프롬프트 버전을 명시해 재현성 확보
- 스킬 변경 시 관련 명세 문서 링크를 같이 업데이트
