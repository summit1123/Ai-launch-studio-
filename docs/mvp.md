# MVP 범위

## 1. MVP 정의
제품 보유 사용자가 대화만으로 아래 결과를 얻는 것
- 시장/포지셔닝 평가
- 주간 홍보 실행 플랜
- 카피/포스터/영상+음성 패키지

## 2. 핵심 사용자
- 이미 제품을 판매하거나 출시한 사용자
- 빠르게 홍보 방향과 실행 소재가 필요한 사용자

## 3. 필수 기능 (Must)
- 챗봇 기반 브리프 수집 + 스트리밍 응답
- 조건부 질문으로 슬롯 채우기
- 브리프 게이트 통과 후 전략 생성
- 카피/포스터/영상 프롬프트 생성
- 보이스 스크립트/TTS 설정 생성
- 실행 이력 저장/재조회

## 4. 권장 기능 (Nice)
- 포스터 3종 변형
- 톤 재생성 버튼
- 15초/30초 보이스 스크립트 동시 생성
- 결과 버전(v1/v2) 저장

## 5. 제외 기능 (Not-now)
- 광고 플랫폼 자동 집행
- 재고/매출 시스템 연동
- 외부 데이터 기반 정교한 시장 리서치 자동화

## 6. 브리프 게이트
전략 생성 전 아래 4묶음 충족 필수
1. 제품: 이름/카테고리/특징 3개/가격대
2. 타겟: 누가/왜
3. 채널: 1~2개
4. 목표: 주간 목표 1개

## 7. 결과 화면
1. 요약: 브리프 + 다음 액션
2. 전략: 시장평가/포지셔닝/주간 플랜
3. 소재: 카피/포스터/영상+보이스

## 8. 완료 기준
- 템플릿 폼 없이 대화만으로 1회 완주 가능
- 게이트 미충족 시 다음 단계 차단
- 최소 1세트 이상의 소재 생성 및 확인 가능
