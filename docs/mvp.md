# MVP 범위

## 1. MVP 정의
제품 보유 사용자가 대화만으로 아래 결과를 얻는 것
- 시장/경쟁/채널 스냅샷
- 포지셔닝/메시지 옵션
- 주간 홍보 실행 플랜
- 카피/포스터/영상(+보이스 스크립트) 패키지

## 2. 핵심 사용자
- 이미 제품을 판매/출시한 사용자
- 빠르게 홍보 방향과 실행 소재가 필요한 사용자

## 3. 핵심 사용자 질문
- 우리 제품을 이번 주에 어떻게 홍보할까?
- 경쟁사와 비교했을 때 지금 어떤 메시지가 먹힐까?
- 지금 바로 집행 가능한 카피/포스터/영상/보이스를 어떻게 만들까?

## 4. 필수 기능 (Must)
- 챗봇 기반 브리프 수집 + 스트리밍 응답
- 멀티턴 텍스트 슬롯 누적 수집(LLM+규칙 병합)
- 조건부 질문으로 슬롯 채우기
- 브리프 게이트 통과 후 리서치 실행
- 리서치 반영 전략 생성
- 카피/포스터/영상 프롬프트 생성
- 보이스 스크립트 생성
- 실행 이력 저장/재조회

## 5. 권장 기능 (Nice)
- 포스터 3종 변형
- 톤 재생성 버튼
- 15초/30초 보이스 스크립트 동시 생성
- 결과 버전(v1/v2) 저장

## 6. 제외 기능 (Not-now)
- 광고 플랫폼 자동 집행
- 재고/매출 시스템 연동
- 고도화된 실시간 BI 대시보드

## 7. 브리프 게이트
전략 생성 전 아래 4묶음 충족 필수
1. 제품: 이름/카테고리/특징 3개/가격대
2. 타겟: 누가/왜
3. 채널: 1~2개
4. 목표: 주간 목표 1개

## 8. 상태 흐름 (MVP)
`CHAT_COLLECTING -> BRIEF_READY -> RUN_RESEARCH -> GEN_STRATEGY -> GEN_CREATIVES -> DONE`

## 9. 결과 화면 (실행형 패키지)
1. 요약
- 브리프 정리
- 누락 리스크
- 다음 액션

2. 시장/전략
- 시장 시그널/경쟁사/채널 인사이트
- 포지셔닝 2~3안
- 주간 실행 체크리스트

3. 소재
- 광고 카피
- 포스터 스펙/이미지
- 영상 프롬프트
- 보이스 스크립트/자막/오디오

4. 성과 계획
- KPI
- A/B 테스트 항목
- 다음 대화에서 갱신할 변수

## 10. 완료 기준
- 템플릿 폼 없이 텍스트 입력만으로 1회 완주 가능
- 게이트 미충족 시 다음 단계 차단
- 리서치 결과가 전략 생성에 반영됨
- 최소 1세트 이상의 소재 생성 및 확인 가능
