# AI Launch Studio PRD (MVP 재정의)

## 1. 제품 정의

### 1.1 한 줄 소개
제품 보유자가 대화만으로 시장성을 점검하고, 홍보 실행안과 콘텐츠를 얻는 프로모션 코파일럿

### 1.2 핵심 사용자
이미 제품을 보유한 판매자/브랜드 운영자/소규모 마케팅 팀

### 1.3 해결하려는 문제
- 어떤 메시지와 채널로 홍보할지 결정이 어렵다.
- 전략과 소재 제작이 분리되어 실행 속도가 느리다.
- 영상은 만들었지만 음성/사운드 품질이 떨어져 완성도가 낮다.

## 2. 목표 / 비목표

### 2.1 목표 (MVP)
- 대화형 플래너 수집(텍스트 우선, 음성 확장 지원)
- 브리프 필수 정보 게이트 통과 로직
- 시장평가 + 포지셔닝 + 주간 운영 플랜 생성
- 카피/포스터/영상 프롬프트 + 보이스 스크립트 생성
- 결과 저장 및 기본 다운로드

### 2.2 비목표 (MVP)
- 광고 집행 API 자동 연동
- 재고/매출 시스템 연동
- 완전 자동 BI 리포트

## 3. 핵심 가치
- 실행 전환 시간을 단축한다.
- 전략과 크리에이티브를 하나의 워크플로로 묶는다.
- Voice Agent로 영상 결과물 체감 품질을 높인다.

## 4. 사용자 플로우
1. 사용자 시작: "내 제품 홍보 도와줘"
2. 플래너가 부족한 정보만 조건부 질문
3. 게이트 통과 시 전략 에이전트 실행
4. 크리에이티브/보이스 에이전트 실행
5. 결과 탭에서 복사/다운로드

## 5. 브리프 게이트 (필수 슬롯)
- 제품: 이름, 카테고리, 특징 3개 이상, 가격대
- 타겟: 누가 사는지, 왜 사는지
- 채널: 1~2개
- 목표: 주간 목표 1개(조회/문의/구매)

## 6. 에이전트 구성 (MVP)
- 오케스트레이터: 상태/분기/저장/이벤트 전송
- 플래너: 대화 기반 슬롯 수집
- 전략: 시장성/리스크/포지셔닝/실행안
- 크리에이티브: 카피/포스터/영상 프롬프트
- 보이스: 내레이션 스크립트/TTS 설정/자막 타이밍

## 7. MVP 산출물
- 브리프 요약 카드
- 시장 평가(강점/주의/불확실)
- 메시지/포지셔닝 옵션 2~3개
- 이번 주 실행 체크리스트
- 광고 카피 10개
- 포스터 1~3개
- 영상 프롬프트 + 보이스 패키지

## 8. 안전 장치
- 과장/금지표현 필터
- 톤 재생성(친근/프리미엄/짧게/직설)

## 9. BM
- Free: 실행 횟수 제한, 워터마크
- Pro: 실행량 확대, 고급 export, 보이스 프리셋
- Team: 협업 워크스페이스/승인/이력 관리

## 10. 성공 지표
- 첫 실행 플랜 생성 시간 3분 이내
- 브리프 게이트 통과율 70% 이상
- 결과물 내보내기 완료율 60% 이상
- 사용자 만족도 4.0/5.0 이상
